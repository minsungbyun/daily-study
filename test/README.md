## 문제 올리는 방법
___
- 문제를 낸 후 `add` > `commit` > `push` 합니다.
- 아래의 양식에 맞게 해주세요 ~
```
// name   : origin 
// branch : main

// add
git add .

// commit
git commit -m "21/07/06 (해당날짜) 문제 출제 완료 (이름)"

// push
git push
git push -u origin main
```

- 다른 기능도 연습해봅시다 ~
